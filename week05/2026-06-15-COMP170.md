## Today's Problem

We set out to write a program that takes a sentence — `"The relentless summer heat in Chicago made even the sidewalks shimmer"` — and produces a list of every word longer than a given number of characters, converted to uppercase. With Bella's help we identified the qualifying words when the cutoff is 4: *relentless, summer, Chicago, sidewalks, shimmer*.

## Splitting a String into Words

The key tool is `.split()`. Because a string in Python is an object — data with built-in behavior — we can write `words = sentence.split()`, and Python breaks the sentence into a list of individual words wherever it finds a space. We also demonstrated splitting on a custom character: splitting a DNA sequence on `"G"` (guanine) chops it into the segments between guanine bases.

## Building the Solution

With `words` in hand, we looped `for i in range(len(words))`, checked `if len(words[i]) > n`, and, for qualifying words, called `.upper()` to convert to uppercase. We also showed how to compute the conversion from scratch using ASCII values (uppercase and lowercase letters are 32 apart), then showed that `.upper()` does the same thing in one step. Finally, we replaced the print statement with an empty output list and `.append()`, collecting results before printing the full list at the end.
